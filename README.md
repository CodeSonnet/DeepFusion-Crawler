# 📱 跨平台智能手机口碑舆情爬虫与决策分析系统

本项目是一个专注于收集主流头部电商平台（京东、淘宝/天猫）智能手机用户真实口碑数据的 Python 爬虫项目，旨在为跨平台数据融合、情感分析及消费者多准则决策（MCDM）提供高质量的数据支持（国家级大学生创新创业训练计划核心模块）。

项目目前集成了对 **京东 (JD)** 和 **淘宝/天猫 (Taobao/Tmall)** 两大平台的高价值评价数据（含追评、点赞数、多图链接）的定向爬取。

## ✨ 主要功能

* **平台覆盖核心电商**: 支持国内最大的两大 3C 数码交易阵地（京东、淘宝/天猫）。
* **核心数据全维提取**: 精准抓取商品评价的关键字段，包括评论ID、发布日期、打分、评论正文、购买SKU、追加评论（追评）、点赞数和买家秀原图链接等。
* **硬核反爬策略应对**: 针对不同电商平台的风控特点采用了降维打击的对抗策略：
    * **京东**: 弃用传统且不稳定的 DOM 解析，采用 `DrissionPage` 拦截底层 `client.action` API 数据包。独创**“队列清空模式 (Queue Draining)”**，无视前端页面滑动卡顿，确保数据包 0 漏抓。
    * **淘宝/天猫**: 采用动态挂载本地浏览器 `User Data` 实现登录状态持久化，完美绕过阿里盾滑块验证；利用正则表达式精准剥离 `mtopjsonp` 跨域外壳，还原核心 JSON 数据。
* **配置简单，自动化轮询**: 用户只需在脚本末尾修改目标商品的 URL 列表，程序即可自动实现多店铺的无缝跳转、动态休眠防封与断点续传。
* **结构化与标准化输出**: 所有平台爬取的结果均经过统一的字段 Mapping 处理，自动保存为标准 `.json` 格式文件，直接赋能后续的 Pandas 清洗与深度学习模型训练。

## 🛠️ 技术栈

* **Python 3.x**
* **核心爬取库**:
    * `DrissionPage`: 负责浏览器底层环境接管、网络包监听（Packet Interception）与自动化动作模拟。
    * `json` & `re`: 负责复杂网络响应体的序列化与正则表达式解析解包。
* **数据存储**: 标准 `JSON` 格式

## 🚀 快速开始与配置说明

使用前请确保已安装核心依赖库：
```bash
pip install DrissionPage
```
### 1. 京东商城爬虫 
此脚本通过监听京东商品详情页的 client.action 接口，结合动态长短休眠策略，实现多店铺的无缝轮询抓取。

* **如何配置:**
打开 spider_jd.py 文件，找到顶部的 ⚙️ 配置区域，按需修改产品名称和店铺链接列表：

```Python
# ================= ⚙️ 配置区域 =================
PRODUCT_NAME = "iPhone 17 Pro"  # 车型/机型名称，用于保存字段标记

# 目标机型的京东店铺链接列表 (请确保每个URL后有逗号)
SHOP_URLS = [
    "[https://item.jd.com/100066896918.html](https://item.jd.com/100066896918.html)", # 京东自营店
    "[https://item.jd.com/100209267859.html](https://item.jd.com/100209267859.html)", # 官方旗舰店
]

# 爬取结果的保存路径
JSON_FILE = 'data/jd_iphone17_pro.json'
# ===============================================
```
* **如何运行:**
```
Bash
python spider_jd.py
```
* **输出文件:**
脚本运行结束后，将在 data/ 目录下生成相应的 .json 数据集文件。

### 2. 淘宝/天猫爬虫 
此脚本监听天猫底层 mtop.taobao.rate.detaillist.get 接口。由于淘宝风控极其严格，脚本强制引入了本地浏览器环境持久化机制以继承真实的账号 Cookies。

* **如何配置:**
打开 spider_taobao.py 文件，提取精简版的商品 ID 并配置环境路径：

```Python
# ================= ⚙️ 配置区域 =================
PRODUCT_NAME = "iPhone 17 Pro"

# 提取淘宝/天猫链接中的纯数字 id 参数即可 (务必加逗号)
SHOP_URLS = [
    "[https://detail.tmall.com/item.htm?id=974199527610](https://detail.tmall.com/item.htm?id=974199527610)", 
]

JSON_FILE = 'data/taobao_iphone17_pro.json'

# 指定一个本地文件夹用于保存淘宝登录环境 (首次运行会自动生成)
USER_DATA_DIR = './Taobao_User_Data'  
# ===============================================
```
* **⚠️ 首次运行极其重要:**
执行 python spider_taobao.py 后，程序会自动打开淘宝首页并暂停。请在弹出的浏览器中务必手动扫码登录。登录成功并在控制台按下回车后，程序才会带着合法的身份凭证开始自动化抓取。后续再次运行将自动读取 USER_DATA_DIR，无需反复登录。

* **如何运行:**

```Bash
python spider_taobao.py
```
