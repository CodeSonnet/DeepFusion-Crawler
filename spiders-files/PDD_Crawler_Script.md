## 功能简介

本爬虫可自动完成以下操作：
1. 根据关键词搜索拼多多商品
2. 获取商品ID
3. 批量抓取商品评论（支持翻页）
4. 自动保存为JSON格式

## Cookie获取

Cookie 是访问拼多多API的必要凭证，需要从浏览器开发者工具获取。

### 获取步骤

1. 打开 Chrome/Edge 浏览器，新建无痕模式窗口
2. 访问 [拼多多搜索页](https://mobile.yangkeduo.com/)
3. **登录**你的拼多多账号（推荐使用在手机APP登录的账号扫码登陆）
4. 按 `F12` 打开开发者工具
5. 切换到 **Network（网络）** 标签
6. 在搜索框输入任意关键词（比如`Cookie`）进行搜索
7. 在网络请求中找到任意一个请求（通常是 `v5/goods` 或类似API请求）
8. 点击该请求，在 **Headers** 中找到 **Request Headers**
9. 复制完整的 `Cookie` 字段全部内容


### Cookie 格式

获取的 Cookie 类似以下格式：

```
api_uid=Ck/ucWnF98u5dACFELgGAg==; _nano_fp=***; webp=1; jrpl=***; njrpl=***; dilx=***; PDDAccessToken=***; pdd_user_id=***; pdd_user_uin=***; pdd_vds=***
```

### 更新Cookie

打开 `PDD_Crawler_Script.py`，找到第17行：

```python
COOKIE = """你的Cookie内容"""
```

将获取的完整Cookie字符串替换即可。

## 配置说明

### 脚本内配置区域（第16-23行）

```python
# ==================== 配置区（从F12复制Cookie） ====================
COOKIE = """你的Cookie"""

HEADERS = {
    "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 ...",
    "Referer": "https://mobile.yangkeduo.com/",
    "Cookie": COOKIE,
}
```

### 可调整参数

| 参数 | 位置 | 说明 | 默认值 |
|------|------|------|--------|
| `max_goods` | 第80行 `search_and_get_reviews` | 最多抓取几个商品的评论 | 10 |
| `max_pages_per_goods` | 第80行 | 每个商品最多翻几页 | 50 |
| 延迟时间 | 第139、144行 | 请求间隔（秒） | 0.2-0.5 / 0.5-1.5 |

## 输出说明

### 保存位置

```
{项目根目录}/data/Pinduoduo/pdd_reviews_{关键词}.json
```

### 数据字段

每条评论包含以下字段：

| 字段 | 说明 |
|------|------|
| `id` | 评论唯一标识 |
| `content` | 评论内容 |
| `score` | 评分星级 |
| `date` | 评论时间 |
| `model_sku` | 商品规格型号 |
| `append_content` | 追评内容 |
| `votes` | 点赞数 |
| `images` | 评论图片列表 |
| `reply` | 商家回复 |
| `platform` | 平台（固定为"Pinduoduo"） |
| `product_name` | 商品名称 |
| `goods_id` | 商品ID |

## 注意事项

1. **Cookie有效期**：Cookie通常有有效期限制，若爬取失败请重新获取Cookie
2. **请求频率**：脚本内置随机延迟，请求间隔0.2-1.5秒，请勿修改过小以免被封
3. **Cookie安全**：请勿将你的Cookie泄露给他人
4. **数据用途**：请遵守拼多多平台协议，合理使用数据
5. **异常处理**：若遇到403/429等错误，可能是Cookie失效或触发了风控，请更换Cookie或稍后再试
