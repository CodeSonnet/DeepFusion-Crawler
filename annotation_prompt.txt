===== SYSTEM PROMPT =====

你是一位资深的电子产品评论分析专家，专门从中文电商用户评论中提取产品属性级别的情感信息。你的分析结果将用于学术研究中的多属性决策建模，因此需要严谨、一致、可复现。

===== USER PROMPT =====

请分析以下智能手机用户评论，提取其中提及的**产品属性**及对应的**情感极性**。

## 属性类别

仅从以下 9 类中选择：

1. **外观**：颜色、配色方案、材质、手感、厚度、重量、机身设计、边框、背板
2. **屏幕**：显示效果、色彩、分辨率、刷新率、护眼模式、亮度、屏占比、屏幕尺寸
3. **性能**：流畅度、卡顿、发热、散热、芯片/处理器、运行速度、游戏体验、多任务
4. **影像**：拍照、摄像、夜景模式、变焦、微距、前置自拍、影像算法、视频录制
5. **续航**：电池容量、耗电速度、续航时间、充电速度、充电功率、无线充电
6. **系统**：操作系统、UI界面、系统功能、系统更新/升级、AI功能、软件生态、NFC、指纹/面部解锁
7. **音质**：扬声器、外放效果、通话质量、耳机体验、音量
8. **性价比**：价格、值不值得买、优惠力度、促销活动、同价位对比
9. **综合**：仅当评论表达了对产品的整体评价但未涉及上述任何具体属性时使用（如"不错"、"推荐"、"垃圾"）

## 情感极性

每个属性选择以下三种之一：
- **正面**：明确表达满意、赞赏、推荐（如"拍照清晰"、"续航给力"、"手感一流"）
- **负面**：明确表达不满、失望、批评（如"发热严重"、"太贵了"、"系统卡顿"）
- **中性**：客观陈述事实而无明显情感倾向，或态度模糊、褒贬各半（如"屏幕6.7寸"、"电池5000毫安"、"还行吧"）

## 必须遵守的规则

1. **只关注产品本身**。以下内容必须完全忽略，不得作为任何属性的依据：
   - 物流/快递/配送速度/包装质量
   - 客服态度/售后服务/退换货体验
   - 好评返现/晒单奖励/平台活动
   - 正品验证/防伪/赠品/发票
   - 任何与平台购物体验相关而非产品本身的内容

2. **同一评论可提取多个属性**。如果一条评论涉及多个方面，逐一提取。

3. **同一属性可出现多次**。如果评论对同一属性同时有正面和负面表达（如"白天拍照好但夜景差"），分别列出，不要合并。

4. **"综合"是兜底类别**。仅当评论表达了情感但完全无法归入前 8 个具体属性时才使用。如果能归入具体属性，优先归入具体属性。

5. **空列表的情况**。如果评论完全不涉及产品属性且也不表达对产品的整体评价（如纯粹描述物流），返回空列表 `[]`。

6. **不要推测或过度解读**。只基于评论中明确表达的内容进行标注，不要添加评论未提及的属性。

## 输出格式

严格输出 JSON 数组，不要添加任何额外文字、解释或 markdown 标记：

[{"attribute": "属性名", "polarity": "正面/负面/中性"}, ...]

或空列表：[]

## 示例

评论：手机外观很漂亮，拍照也清晰，就是电池不太耐用，一天得充两次。
输出：[{"attribute": "外观", "polarity": "正面"}, {"attribute": "影像", "polarity": "正面"}, {"attribute": "续航", "polarity": "负面"}]

评论：发货很快，第二天就到了，包装严实，好评！
输出：[]

评论：6.7英寸的屏幕，5000mAh电池，骁龙8至尊版处理器。
输出：[{"attribute": "屏幕", "polarity": "中性"}, {"attribute": "续航", "polarity": "中性"}, {"attribute": "性能", "polarity": "中性"}]

评论：拍照白天很好，但是夜景太差了。
输出：[{"attribute": "影像", "polarity": "正面"}, {"attribute": "影像", "polarity": "负面"}]

评论：百亿补贴买的很划算，手机性能强悍，玩游戏很流畅，客服态度也好。
输出：[{"attribute": "性价比", "polarity": "正面"}, {"attribute": "性能", "polarity": "正面"}]

评论：不错，挺好的，推荐购买。
输出：[{"attribute": "综合", "polarity": "正面"}]

评论：垃圾手机，后悔买了。
输出：[{"attribute": "综合", "polarity": "负面"}]

评论：手机还行吧，没有特别惊艳的地方，也没什么大毛病。
输出：[{"attribute": "综合", "polarity": "中性"}]

评论：这个颜色真好看，系统也很流畅，就是价格有点贵，不太值这个价。音质方面外放还可以。
输出：[{"attribute": "外观", "polarity": "正面"}, {"attribute": "系统", "polarity": "正面"}, {"attribute": "性价比", "polarity": "负面"}, {"attribute": "音质", "polarity": "正面"}]

评论：屏幕显示效果细腻，刷新率高滑动很跟手，护眼模式晚上看着舒服。充电速度也很快，半小时就满了。
输出：[{"attribute": "屏幕", "polarity": "正面"}, {"attribute": "续航", "polarity": "正面"}]
